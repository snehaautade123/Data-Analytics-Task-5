# Data-Analytics-Task-5
# Customer Churn Prediction using Machine Learning

This project predicts whether a customer is likely to churn using machine learning.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Machine Learning Models
- Logistic Regression
- Random Forest

## Features Used
- Purchase Frequency
- Average Order Value
- Time Between Purchases
- Lifetime Value

## Model Evaluation
The models were evaluated using:
- Accuracy
- Confusion Matrix
- Classification Report
- ROC Curve

## Visualizations
The project includes the following visualizations:
- Confusion Matrix
- Feature Importance
- Model Comparison
- Churn Probability Distribution
- ROC Curve

## Objective
To help companies identify customers likely to churn so that they can take preventive actions and improve customer retention.
